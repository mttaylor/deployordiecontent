# EP007 — AI Newsletter Draft Agent

From Deploy or Die Episode 7: "I Set Up an AI Agent to Draft My Weekly Newsletter. Here's the Prompt."

## Files

- `system_prompt.md` — set this as your system prompt in any Claude/GPT session
- `draft_prompt.md` — run this each week with your topic filled in

## How to use

1. Open Claude or ChatGPT
2. Paste `system_prompt.md` as the system prompt
3. Fill in `draft_prompt.md` with your topic and run it
4. Edit for 15 minutes — fix anything that sounds off, add real examples
5. Ship Thursday
